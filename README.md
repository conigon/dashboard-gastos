# 💰 Dashboard de Análisis de Gastos

Dashboard interactivo para analizar gastos mensuales hecho con Python.

## ✨ Funcionalidades
- **Filtro por Mes**: Ver gastos de Enero, Febrero, Marzo, Abril
- **Vista Acumulada**: Ver el total de todos los meses
- **Filtros por Categoría**: Arriendo, Comida, Transporte, etc.
- **3 KPIs**: Gasto Total, Promedio por Gasto, Nº de Gastos
- **3 Gráficos**: Barras por categoría, Pie de distribución, Línea de tendencia mensual

## 🛠️ Tecnologías Usadas
`Python` `Streamlit` `Pandas` `Plotly`
